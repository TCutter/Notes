## Git
概念：分布式版本控制系统
版本控制：对代码、文档的变更进行管理

### 集中式 VS 分布式

#### 集中式
版本库集中放置在中央服务器
![集中式](../Style/images/git/centralize.png)

#### 分布式
每个人的电脑都有完整的版本库
![分布式](../Style/images/git/hadoop.png)
